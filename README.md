📚 Library Management System
A modern Library Management System designed to efficiently manage books, students, and borrowing activities. This project helps automate library operations like issuing books, tracking returns, and maintaining records.
🚀 Features
📖 Add, edit, delete books
👨‍🎓 Manage student records
🔍 Search books by title, author, or ISBN
📥 Issue books to students
📤 Return books with tracking
🗂️ View all issued and available books
✏️ Edit and delete records easily
📊 Organized database structure
